1. add -> добавить строку
2. remove -> удалить строку
3. Options -> Update Обновить(подкачать данные из файла)
4. Сортировка - Food, Transport, Clothes, Needful, Others, All
5. File -> Create new file - Создать новый файл(пока на этапе разработки)
6. File -> Rename file - Переименовать файл
7. File -> View file - Выбрать проматриваемы файл
8. Option -> Exit - Закрыть программу
Справа снизу отображается сумма отображаемых строк(столбец Amount)
Дату вводить желательно в следующем формате - 14.11.2020
Сумму вводить без точек и доп.символов
